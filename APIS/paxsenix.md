# PaxSenix API

PaxSenix API is a free REST API service providing access to various AI functionalities including chat, persona, search, image generation, text-to-video, text-to-music, and more. It offers a comprehensive set of AI tools with a rate limit of 25 requests per minute.

## Pros

- Wide range of AI services in one API
- Free to use with API key
- Supports multiple AI modalities

## Cons

- Rate limited to 25 requests per minute
- Requires API key registration
